# Dr. Alok Misra - Academic Portfolio

A professional academic portfolio and evaluation portal built for Dr. Alok Misra, Professor of Computer Science. 

This platform serves as a comprehensive digital presence, showcasing research outputs, academic experience, and insights, while also providing a dedicated, streamlined portal for student project and patent submissions.

## Key Features

- **Academic Profile:** Detailed sections highlighting core research areas, professional experience, and educational background.
- **Publications & Insights:** A curated list of scholarly works and an academic blog for sharing research insights.
- **Evaluation Portal:** A secure, custom-built submission portal (powered by PatentIQ | DiaLabs) enabling students to submit projects and patent drafts for review.
- **Modern & Responsive:** A clean, accessible, and fully responsive user interface built with modern web technologies.

## Technology Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** Custom CSS with a polished, minimalist design system
- **Backend/API:** Integrated with DiaLabs worker APIs for submission handling
